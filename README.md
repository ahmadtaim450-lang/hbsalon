# HB Beauty Salon - دليل التشغيل

## الملفات

1. **`index.html`** - الموقع الرئيسي (للزبائن)
2. **`login.html`** - صفحة تسجيل دخول الأدمن
3. **`dashboard.html`** - لوحة التحكم (للأدمن)
4. **`firestore.rules`** - قواعد الأمان (مهمة جداً!)

---

## خطوات الإعداد (مرة واحدة فقط)

### 1. تفعيل Firebase Authentication
1. اذهبي إلى: https://console.firebase.google.com/project/hayasalon-f2cd1/authentication
2. اضغطي على **"Get Started"**
3. اختاري **"Email/Password"** من قائمة موفري تسجيل الدخول
4. فعّلي الخيار الأول (Email/Password)
5. احفظي

### 2. تفعيل Firestore Database
1. اذهبي إلى: https://console.firebase.google.com/project/hayasalon-f2cd1/firestore
2. اضغطي **"Create database"**
3. اختاري **"Start in production mode"**
4. اختاري المنطقة (مثل `europe-west3` - فرانكفورت)
5. انتظري حتى ينتهي الإعداد

### 3. تطبيق قواعد الأمان
1. اذهبي إلى: https://console.firebase.google.com/project/hayasalon-f2cd1/firestore/rules
2. افتحي ملف `firestore.rules` وانسخي المحتوى من **`rules_version = '2';`** حتى النهاية
3. الصقيه في Firebase Console
4. اضغطي **"Publish"**

### 4. إنشاء حساب الأدمن الأول
1. افتحي `login.html` في المتصفح
2. اضغطي **"Konto erstellen"** (إنشاء حساب)
3. أدخلي بريد إلكتروني وكلمة مرور (6 أحرف على الأقل)
4. ستدخلين تلقائياً إلى الداشبورد

## كيف يعمل النظام

### للزبائن (الموقع الرئيسي):
- يملأ الزبون نموذج الحجز
- الحجز يُخزن في Firebase بحالة **"pending"** (معلق)
- الزبون يرى رسالة "تم إرسال طلبك بنجاح"

### للأدمن (الداشبورد):
- **3 تبويبات:**
  - **Anfragen (الأنفعاد):** الطلبات المعلقة فقط - تظهر أولاً
  - **Alle (الكل):** كل المواعيد مع فلاتر
  - **Kalender (الروزنامة):** عرض شهري للمواعيد

- **لكل موعد:** زر بستغيد + زر رفض + زر تفاصيل
- **الداشبورد في الوقت الفعلي:** إذا أحد حجز الآن، يظهر فوراً

### الروزنامة الذكية:
- **الأحد والاثنين:** مغلقة دائماً
- **الأيام المعطلة من الأدمن:** تظهر بشطب أحمر للزبائن والأدمن
- **المواعيد المؤكدة:** تحجز الوقت تلقائياً (ما يمكن لزبون ثاني يحجز نفس الوقت)
- **إذا كل الأوقات محجوزة ليوم معين:** اليوم يختفي من روزنامة الزبائن
- **الساعات السابقة (اليوم الحالي):** لا تظهر

### تعطيل يوم في الداشبورد:
1. افتحي تبويب **Kalender**
2. اضغطي على أي يوم
3. اضغطي **"Tag sperren"** (تعطيل اليوم)
4. اليوم الآن مقفل للحجوزات الجديدة
